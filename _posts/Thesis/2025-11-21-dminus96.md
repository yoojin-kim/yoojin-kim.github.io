# Mamaba Environment

[1] Create a new environment: 
>conda create -n mamba python=3.10  

[2] activate: 
>conda activate mamba

[3] install pytorch: 
>conda install pytorch==2.4.1 torchvision==0.19.1 torchaudio==2.4.1  pytorch-cuda=11.8 -c pytorch -c nvidia

[4] load cuda module: 
>module load cuda/11.8
